Infineon TPM Firmware Update Demo



Infineon is the first TPM vendor to open source their firmware update procedure and process in their latest Infineon SLB9672 (SPI) and SLB9673 (I2C) versions of the TPM 2.0 module.

wolfTPM is the only library to offer integrated support for updating TPM firmware.

Demo Platform:

  • Infineon PSoC 62S2 evaluation kit (Wifi)
  • Infineon SLB9373 (I2C) TPM 2.0 mikroBUS module
  • Modus Toolbox Wi-Fi-HTTPS-Server demo
    • wolfSSL TLS v1.3 server
    • wolfTPM
